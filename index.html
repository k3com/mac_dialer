OS X Address Book Plugins for IP phones

dial directly form your OS X address book

Mit diesen Plugins kann man direkt aus dem Adressbuch mit IP Telefonen von Snom und Aastra sowie Systemtelefonen an Auerswald Telefonanlagen wählen. Leider nur bis macOS 10.13

Das gewünschte Plugin muss in Library/Application Scripts/com.apple.AddressBook im Benutzerordner kopiert werden.

These plugins allow you to dial numbers directly from your address book. To use them you must copy them to Library/Application Scripts/com.apple.AddressBook (before Mavericks: Library/Address Book Plug-Ins) in your user directory.

Download

  • Aastra
  • Auerswald (aktuelle Modelle)
  • Auerswald bis Basic 2
  • Snom

Installation

Im Finder "Gehe Zu" aufklappen, dann die alt-Taste gedrückt halten und Library auswählen. In den Ordner "Application Scripts" gehend und dort den Ordner "com.apple.AddressBook" anlegen und das gewünschte Plugin dahin kopieren.

Nun AppleScript Datei öffnen und am Anfang die Einstellungen anpassen.

Bei Aastra muss zusätzlich im Telefon selbst unter Configuration Server die IP des Rechners in die Liste ganz unten eingetragen werden.

Nach den Änderungen Adressbuch beenden und neu starten.
